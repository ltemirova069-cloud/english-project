Women Support Website

Help • Safety • Equality

Why this website?

Many women face unfair treatment and violence. Our website shows how women can get help and support. This is a demo project for education.

How it works

A woman can scan a QR code and come to this website. She can report a problem and ask for help. This website shows how support systems can work.

Report a problem

If a woman is treated unfairly, she can write about the situation. The system shows that action will be taken.

Send Report

This is a demo. Data is not saved. The fine is shown as an idea only.

Our message

Women deserve safety, respect, and equal rights. Society grows when women are protected and supported.

Educational project • No real data • No real fines
